# 🚀 Smart Health Monitor

Real-time health monitoring dashboard built with Python Flask and Chart.js for visualizing live patient data and health metrics.

![Project Status](https://img.shields.io/badge/Status-Completed-success)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🌟 Features

- [x] Real-time vital signs monitoring dashboard\n- [x] Animated live Chart.js graphs for heart rate & SpO2\n- [x] REST API backend with Python Flask

## 🛠️ Tech Stack

- **Python**\n- **Flask**\n- **Chart.js**\n- **JavaScript**

## 🚀 How to Download & Run

Follow these steps to set up the project on your local machine.

1. Clone the repository:
   ```bash
   git clone https://github.com/Aliffhafizirosdi/smart-health-monitor.git
   ```
2. Navigate to the directory:
   ```bash
   cd smart-health-monitor
   ```
3. Create a virtual environment and install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
5. Open your browser and navigate to `http://127.0.0.1:5000`
